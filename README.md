# sql-data-analytics-project
This projects demonstrates a production-ready approach to data analysis using advanced SQL. It features a collection of optimized scripts designed to solve real-world business problems, ranging from exploratory data analysis (EDA) to complex KPI reporting.

Key Technical Highlights:

Advanced Logic: Utilization of Window Functions, CTEs, and self-joins for multi-dimensional analysis.

Business Intelligence: Scripts for cohort analysis, customer segmentation (RFM), and retention tracking.

Performance: Queries structured for readability, scalability, and efficient execution on large datasets.
